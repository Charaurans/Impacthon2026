# 🤝 Contribuir al Proyecto

¡Gracias por tu interés en mejorar esta herramienta! Para mantener el orden, sigue estas pautas:

## Cómo empezar
1. Haz un **Fork** del repositorio.
2. Crea una rama para tu mejora: `git checkout -b feature/nueva-mejora`.
3. Realiza tus cambios y haz un **Commit**: `git commit -m 'Añade nueva funcionalidad'`.
4. Envía un **Pull Request**.

## Estilo de Código
* Mantener la lógica de renderizado en `main.js`.
* Usar nombres de variables descriptivos.
* Documentar funciones críticas.

## Reportar Errores
Si encuentras un bug, por favor abre un **Issue** detallando los pasos para reproducirlo y el error que aparece en la consola.
